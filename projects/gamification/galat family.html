Galat Family


A project by IDC, IIT B along with RSK (Rajya Shiksha Kendra) for government schools of Madhya Pradesh on 'The Gamefication of Education'. Involving the creation of supplementary material that is game based and interactive in order to enhance the quality of learning in primary schools across the state. (class 1 -5)

The idea of creating such material would be to engage the children and help them learn in a playful manner particularly in subjects like Math and language. These could be online, offline or activity based, designed to be played alone, in a group or with the teacher. They would have to be thematically integrated, contextual and a story running through may even integrate teaching both math and language in the same game.The difficulty level could be increased and levels created of the same game for higher classes.

The game comprises of 52 set of hand cards corresponding to 52 letter from hindi varnmala. Each set contains about 3 correct letters, 2 wrong version of the same letter, images associated with selected letter and a bonus cartoon character Each card is color coded, set wise on its back To organize them in a set 52 large card with correct letters for reference and for selecting letter during play Difficulty can be set by the players themselves by adding more than one set at a time
